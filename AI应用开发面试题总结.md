## 参考答案要点

### 熔断器实现要点
```python
class CircuitBreaker:
    def __init__(self, failure_threshold=5, recovery_timeout=30):
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self._status = defaultdict(lambda: CircuitBreakerStatus.CLOSED)
        self._failure_counts = defaultdict(int)
        self._last_failure_time = defaultdict(float)
    
    def can_execute(self, agent_name: str) -> bool:
        status = self.get_status(agent_name)
        if status == CircuitBreakerStatus.OPEN:
            # 检查是否超时，超时则转为HALF_OPEN
            if time.time() - self._last_failure_time[agent_name] >= self.recovery_timeout:
                self._status[agent_name] = CircuitBreakerStatus.HALF_OPEN
                return True
            return False
        return True
```

### RAG优化要点
1. **Query改写**：使用LLM扩展查询、提取关键词
2. **混合检索**：向量相似度 + BM25关键词匹配
3. **重排序**：Cross-encoder精排、学习排序（LTR）
4. **缓存优化**：Query缓存、Embedding缓存、结果缓存

### 性能优化要点
1. **异步化**：全链路异步、线程池隔离
2. **流式输出**：SSE实时推送、增量渲染
3. **预加载**：热门数据预热、模型预加载
4. **批量处理**：请求合并、批量Embedding
